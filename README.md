Installation
--------------
Install the following applications:

* stud.io studio
* LPub3D
* LDView
* LDDraw

Process
-----------
* Create inventory in stud.io studio
* Export inventory to ldr
* Import ldr in LPub3D
* Add BOM
* Export BOM to PDF 
* Open PDF in Illustrator and scale down to 50%
* Layout the parts
* Save PDF and print
